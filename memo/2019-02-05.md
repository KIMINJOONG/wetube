오늘의 오류
==================================
mongoose에서 findOneAndUpdate({ id })만 했더니 업데이트가 안됐다<br/>
findOneAndUpdate( { _id : id})로 해주기!

const videos = await Video.find({}).sort({_id: -1}); ??
=============================================
_id: -1 을했을때 위 아래 순서를 바꾸겠다는 약속과 같다. 지금은 id 순서대로 정렬을 했음<br/>
